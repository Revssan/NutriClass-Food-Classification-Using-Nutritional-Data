**NutriClass: Food Classification Using Nutritional Data**

**Project Overview**

NutriClass is a Machine Learning project that classifies food items based on their nutritional values such as calories, proteins, carbohydrates, fats, and sugar.
The system predicts the exact food name using tabular nutrition data, enabling strict diet planning and intelligent meal selection.

**🎯 Problem Statement**

In today’s health-conscious world, accurately identifying food types based on nutritional composition is essential.
This project develops a multi-class classification model that:
Takes nutritional metrics as input
Predicts the exact food name
Supports strict diet control and automation
